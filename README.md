# Arkanoid

Клон игры Arkanoid (Famicom/NES, 1986) на чистом HTML5 Canvas + CSS3 + JavaScript (ES-модули),
без сборщиков, фреймворков и игровых движков.

Референс: [Arkanoid (Famicom, 1986) — 1P Full Game Session](https://rutube.ru/video/8ddb5daab8186764ed2ef495eb326740/?t=78&r=plwd)

## Запуск

```
npx serve .
# либо
python -m http.server 8080
```

Затем открыть `http://localhost:<порт>/index.html` в браузере.

## Управление

| Клавиши             | Действие                                                     |
| ------------------- | ------------------------------------------------------------ |
| `←` `→` или `A` `D` | Движение ракетки                                             |
| `Space`             | Старт / запуск мяча / подтверждение на экранах               |
| `P`                 | Пауза / снятие с паузы                                       |
| `Esc`               | На экране инструкций — назад в меню; на паузе — выход в меню |

При потере окном фокуса игра автоматически ставится на паузу.

## Архитектура

Вся логика инкапсулирована в класс `ArkanoidGame` (`js/game.js`) — единственную точку входа,
которую создаёт `js/main.js`. Никаких глобальных переменных, кроме этого создания экземпляра,
код не оставляет.

```
index.html
css/style.css        — оформление страницы, ретро-скейлинг канваса
js/
  main.js             — точка входа: создаёт ArkanoidGame и запускает цикл
  game.js             — ArkanoidGame: игровой цикл, state machine, коллизии, рендер
  constants.js        — все игровые константы и таблицы типов в одном месте
  input.js            — InputManager: клавиатурный ввод
  levels.js           — раскладки кирпичей по раундам (чистые данные)
  hud.js              — рендер HUD и центральных сообщений
  particles.js        — ParticleSystem: частицы при разрушении кирпичей
  utils.js            — clamp, choice
  entities/
    paddle.js          — Paddle
    ball.js             — Ball
    brick.js            — Brick, BrickField
    powerup.js          — PowerUp
```

Игровой цикл использует fixed-timestep с аккумулятором (60 обновлений/сек), развязанный от
переменного `requestAnimationFrame`-дельта, с защитой от рывка после сворачивания вкладки.
